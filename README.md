# node-express-mongoose

### 一、介绍

这是一个基于node和express框架以及mongodb数据库开发的一个电商接口

### 二、使用说明

##### ①安装依赖包

```shell
npm install
```

##### ②需要在本地有mongodb数据库

##### ③运行

```shell
npm run serve
```

##### ④接口文档

http://127.0.0.1:3000/public/doc/index.html



### 三、技术栈介绍

①node

②express框架

③mongoose

④md5数据加密

⑤apidoc接口文档

⑥该接口使用了三层架构思想
